# Yellow moon shop
## Brief Introduction
This repository is a web application in FPT University's Web Java Lab.
This is online shopping website for selling moon cake with below function:
* Login (password encrypted using SHA-256)
* Search cake
* Shopping
* Create and update cake
* Tracking order
* Login via third party, in this project is Facebook

This project use Filter as Controller and follow MVC2 model.

## System dependencies
* Microsoft SQL Server 2018
* JavaEE 5
* Tommcat 8.0.27
* JDK8
